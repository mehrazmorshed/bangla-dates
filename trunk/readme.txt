Convert English months and weekdays into Bangla language

== Description ==

This plugin allows you to convert English months and weekdays into Bangla language in your website

== Installation ==

Install the Bangla Dates plugin via WordPress Dashboard (from plugin directory or from your computer), or upload the files manually to your server and activate the plugin.

= Install via WordPress dashboard =

*From Plugin Directory*

* Visit ‘Plugins > Add New’
* Search for ‘Bangla Dates’
* Install and Activate the plugin

*From Your Computer*

* Download the Bangla Dates plugin
* Visit ‘Plugins > Add New > Upload Plugin’
* Choose the plugin file in .zip format
* Click the Install Now button, and Activate after installation

= Install via Uploading Files Manually =

* Download the Bangla Dates plugin and unzip the folder
* Upload the folder to your ‘/wp-content/plugins/’ directory
* Activate the Bangla Dates Plugin from Dashboard

== Frequently Asked Questions ==

= How to enable Bangla dates? =

Activate the plugin. That's it.

= How to get English dates back? =

Just deactivate the plugin.
